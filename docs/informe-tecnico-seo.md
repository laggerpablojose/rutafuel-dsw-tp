## Estrategias SEO aplicadas

### SEO aplicado en Ubicación

En la página `ubicacion.html` se definió un título específico: `Ubicación | RutaFuel`, acompañado de una meta description orientada a la búsqueda de ubicación, horarios y datos de contacto de la estación.

La página mantiene una sola etiqueta `h1`: “Encontranos fácilmente”, lo que permite identificar claramente el tema principal del contenido. Las tarjetas inferiores utilizan encabezados secundarios para conservar una jerarquía ordenada: Ubicación, Horarios y Contacto.

También se aplicaron atributos `alt` descriptivos en las imágenes utilizadas. El mapa principal se describe como imagen de cobertura y ubicación de RutaFuel, mientras que las imágenes de las tarjetas representan visualmente los surtidores, la fachada nocturna y el personal de atención.

La URL `ubicacion.html` es simple, clara y amigable. Además, la página se encuentra enlazada desde el menú principal del sitio y desde otros bloques internos, favoreciendo la navegación entre secciones.

### Síntesis de lo aplicado.

- Uso de etiquetas semánticas: header, nav, main, section, article, aside y footer.
- Una sola etiqueta h1 por página.
- Titles específicos por página.
- Meta descriptions específicas por página.
- URLs amigables con nombres simples.
- Enlaces internos entre las páginas principales.
- CSS compilado y centralizado en css/main.css.
- JavaScript propio separado en js/main.js.
- Preparación para uso de imágenes con atributo alt.
