# RutaFuel

## Descripción

RutaFuel es un proyecto académico desarrollado para la materia Desarrollo de Sitios Web de la Tecnicatura Universitaria en Tecnología de la Información (UTN FRSF).

El proyecto consiste en un diseño y desarrollo de una página web institucional responsive para una estación de servicio ficticia inspirada en modelos modernos de estaciones de servicio actuales

El sitio busca representar una identidad visual moderna, clara y adaptable a múltiples dispositivos, aplicando conceptos de maquetación web, estructura semátnica HTML y diseño responsive.

---

## Objetivo del Proyecto

Desarrollar un sitio web institucional responsive aplicando los conocimientos adquiridos en la materia Desarrollo de Sitios Web, utilizando tecnologías frontend básicas y buenas prácticas de maquetación.

El proyecto tiene como finalidad implementar:

- estructura semántica HTML5
- estilo mediante CSS3
- maquetación con Flexbox y Grid
- diseño responsive mediante Media Queries
- organización visual y jerarquía de contenidos
- documentación técnica y académica

---

## Alcance

El proyecto contempla el análisis, diseño y desarrollo frontend de un sitio web institucional responsive.

Se incluirán:

- estructura y navegación del sitio
- diseño visual y maquetación
- wireframes y prototipos
- implementación responsive
- adaptación multidispositivo
- documentación académica

No se desarrollarán funcionalidades backend ni integración con bases de datos o servicios externos complejos.
